# Transition Net

## Developer Environment

- [Mac OS X](http://www.apple.com/osx/)
- [POW](http://pow.cx/)
- [Git](http://git-scm.com/)

## How to develop

### Checkout and configure

- `$ git clone git@github.com:jmuheim/transition.git`
- `$ cd transition`
- `$ bundle install`
- `$ powder install`
- `$ powder link`

### Develop

- `$ powder open`
- `$ guard`
